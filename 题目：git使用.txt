题目：git使用

一、git配置
1.指定用户名
>git config --global user.name"名称"

2.指定email
>git config --global user.email"邮箱"

二、创建仓库
1.建立文件夹
>mkdir gitwork
2.进行gitwork
>cd gitwork
3.创建仓库
>git init

注:>ls -ah //显示隐藏文件

三、建立文件
在工作目录中，建立readme.txt
>vim readme.txt
注:按ESC切换到命令模式，在命令行中输入：wq

四、将文件存入暂存区(stage)中
工作区文件=>暂存区
>git add readme.txt
>git add .

五、查看当前文件信息
1.查看状态
>git status
2.查看历史
>git log
3.查看不同
>git diff

六、提交文件到仓库
暂存区文件=>仓库
提交文件
>git commit 文件名 -m "说明"

七、从暂存区恢复文件
>git checkout 文件名
>git checkout HEAD 文件名
>git checkout HEAD^ 文件名
>git checkout HEAD~3 文件名

八、从仓库恢复文件
>git reset 文件名(版本号)
注：这只能从仓库恢复到暂存区,并不改变工作区中的内容
要想改变,还要使用
>git checkout 文件名

git reset 系统提供了三个参数
(1) --soft
>git reset --soft版本号：只改变仓库中的内容,暂存区和工作区不受影响。
(2) --mixed
>git reset --mixd 版本号:改变仓库和工作区中的内容,工作区不受影响。
不写参数,默认指的是 --mixed
(3) --hard
>git reset --hard 版本号：改变仓库、暂存区和工作区中的内容/

九、回看操作过程的命令
>git reflog

http://github.com/

3.提交
4.进入邮箱验证注册信息
5.sign in (登录账号)

rsa_id.pub

打开文件,复制所有内容。
3.登录github账号
4.打开setting,找到SSH,进行添加(创建)
十二、添加远程仓库
>git remove add origin https://github.com/AiFengwu/beautiful-hub
或
>git remote add origin git@github.com:AiFengwu/beautiful-hub
十三、查看远程仓库
>git remote
>git remote show 仓库
>git remote -v

十四、推送文件到远程仓库
首先,要使用git pull origin master 完成下载
其次,使用git push origin master完成推送
>git push origin master -f

十五、clone github 仓库
>git clone 仓库地址
